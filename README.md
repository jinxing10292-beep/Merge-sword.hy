# ⚔️ 검 합성 게임 (Merge Sword)

HTML, CSS, JavaScript로 만든 중독성 있는 검 합성 게임!

## 🎮 게임 방법
- 같은 레벨의 검 2개를 드래그하여 합성
- 검을 강화하여 골드 획득량 증가
- 자동으로 골드가 쌓입니다
- 퀘스트를 완료하여 보상 획득

## 🚀 실행 방법
`index.html` 파일을 브라우저에서 열기

## 📝 기능
- 5x5 인벤토리 시스템
- 드래그 앤 드롭 합성
- 검 강화 시스템 (성공/유지/파괴)
- 일일 퀘스트
- 업적 시스템
- 자동 저장
